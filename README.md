# noha-setup

Setup público do ambiente Noha. Não contém segredos — apenas baixa, usando o
login GitHub da própria pessoa (`gh`), as contas de cliente a que ela tem acesso.

**Uso (uma vez):**
```
bash <(curl -fsSL https://raw.githubusercontent.com/NohaConnect/noha-setup/main/noha-setup.sh)
```
Depois: abra o Claude Code dentro da pasta do cliente em `~/Noha/contas/`.
